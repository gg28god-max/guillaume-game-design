Home Run Derby VR

Developed in partnership with MLB for the All-Star Game Championship, Home Run Derby VR is a premium Meta Quest sports simulation. Serving as Lead Architect for the game's UI Visual Identity, the project required engineering a clean, high-performance design system — from typography and color palettes through button components, tournament UIs, first-time user experience flows, progression systems, and Season 3 interface updates.
